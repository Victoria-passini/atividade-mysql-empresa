# 🏢 Banco de Dados - Exercício de Gestão de Empresa

> Script desenvolvido para praticar a criação de tabelas, relacionamentos e inserção de dados simulando um ambiente de RH de uma empresa.

## 🎯 Objetivo
Praticar os fundamentos de bancos de dados relacionais utilizando o MySQL, aplicando regras de chaves primárias e estrangeiras que aprendi em aula.

## 🛠️ Tecnologias Utilizadas
* **Banco de Dados:** MySQL
* **Ferramenta:** IntelliJ IDEA / VS Code

## 📁 Organização do Projeto
Para manter o projeto organizado, estruturei o repositório da seguinte forma:
* `database/queries/company_structure.sql`: Arquivo contendo todo o código SQL do exercício.

## 🚀 Como Executar o Script
1. Abra o arquivo `company_structure.sql` localizado dentro das pastas de database.
2. Copie o código SQL.
3. Cole e execute no seu gerenciador do MySQL (como Workbench ou IntelliJ).

## 🧠 O que Aprendi com este Exercício
* Como relacionar duas tabelas usando `FOREIGN KEY`.
* A importância de planejar quais campos não podem ficar vazios (`NOT NULL`).
* Como organizar os arquivos do banco de dados em pastas no GitHub para não deixar tudo solto.
